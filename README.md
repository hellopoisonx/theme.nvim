# Themes picker for Neovim

## Using it

### Install

Via lazy.nvim

```
{
    'hellopoisonx/themes.nvim',
    opts = {} 
}

```
### Setup

```
opts = {
    ui = {
        dropdown = true|false -- Whether to use dropdown interface
    },
    themes = {} -- string[] theme list
}
```

## Features and structure

- 100% Lua
- Easy to use
